# Exception Handling in Java

A beginner-friendly collection of Java programs covering core exception handling concepts.

## Topics Covered

-Basic try-catch syntax** – catching exceptions and preventing crashes
-Multiple catch blocks** – handling different exception types separately
-try-catch-finally** – ensuring code runs regardless of exceptions
-`finally` with `return` – understanding finally's execution even when return is used in try
-Nested try-catch – handling exceptions inside another try block

## Key Takeaway

Exception handling makes your code robust by gracefully managing runtime errors instead of abrupt crashes. `finally` always executes, multiple `catch` blocks let you handle specific errors differently, and nesting allows fine-grained control in complex logic.
